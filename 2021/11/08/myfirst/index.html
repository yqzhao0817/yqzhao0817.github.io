一、关于博客搭建

1、什么是Hexo

Hexo 是高效的静态站点生成框架，他基于 Node.js。通过 Hexo 你可以轻松地使用 Markdown 编写文章（博主现在使用的编辑器是Typora），除了 Markdown 本身的语法之外，还可以使用 Hexo 提供的 标签插件 来快速的插入特定形式的内容，Hexo，Git，Nodejs具体的安装过程网上有大量的教程，这里就不多说了。

2、使用的一些第三方服务

（1）Algolia Search

Algolia 是一个托管搜索引擎，提供全文，数字和多面搜索，能够从第一次击键中提供实时结果，Algolia强大的API可让您快速无缝地在网站和移动应用程序中实施搜索，感兴趣的话可以去官网自行探索。

（2）文章数据统计（基于 Valine.js 插件）

Valine是一款快速、简洁且高效的无后端评论系统，实现方式：在文章列表页面加载 Valine.js， 只要 Valine 配置项中的 el 元素不存在即不会加载评论框相关内容。 本博客评论功能使用的就是是 Valine.js 插件，感兴趣的话可以去官网自行探索。

（3）赞赏页面

赞赏页面使用了 Sponsor Page，并做了相关修改使其适应博客样式。感兴趣的小伙伴自行去了解吧。
